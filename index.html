Twixtor

Installation Instructions:

1. Open TwixtorV7OFXInstaller folder.

2. Run Twixtor7OFX-7.5.2-windows-installer.exe

3. Complete the installation process. Once you are in the registration key page, just click "Install Demo" and finish.

4. Open Twixtor_7.3.0_OFX folder. Once you see the "Twixtor7OFX" folder, select it and copy it. Paste it in C:\Program Files\Common Files\OFX\Plugins.

5. Agree to overwrite the files and use your plugin in VEGAS PRO.

4. Agree to overwrite the files and use your plugin in VEGAS PRO.
